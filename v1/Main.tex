\paragraph{Instructions:}
\textsc\\
 1. Observe the state of the rack.\textsc\\
 2. If rack has no identifiable issues, such as e.g.\text\\
     \begin{itemize}
         \item open wires
         \item short circuits
         \item missing components
         \item missing fuses
     \end{itemize}
 3. Plug the rack into an available outlet.\textsc\\
 4. Remove the lockout tag (Fig 4).\textsc\\
 5. Energize system by pressing red button (Fig 4).\textsc\\
 6. Toggle H2-DM1E (Fig 5) into run mode from stop.\textsc\\
 7. When rack is energized:
    \begin{itemize}
        \item the Red LED (labeled: Intertie) should be on.
        \item the Red LEDs (labeled: transformer, bus, feeder, and branch in Fig 2) should be off.
    \end{itemize}
 8. Turn on the Transformer by pressing the corresponding Blue Button (Fig 3), which should toggle the Green LED on and the Red LED off (Fig 2).\textsc\\
 9. Turn on the Bus by pressing the corresponding Blue Button (Fig 3), which should toggle the Green LED on and the Red LED off (Fig 2).\textsc\\
 10. Turn on the Feeder by pressing the corresponding Blue Button (Fig 3), which should toggle the Green LED on and the Red LED off (Fig 2).\textsc\\
 11. Turn on the Branch by pressing the corresponding Blue Button (Fig 3), which should toggle the Green LED on and the Red LED off (Fig 2).\textsc\\
 \begin{itemize}
     \item Note: The buttons will only activate in succession
 \end{itemize}
 12. When any blue button is pressed again, the corresponding Green LED should toggle off and the Red LED should turn on again (This should be done in reverse succession).\textsc\\
 13. When de-energizing the rack, toggle the Lock-out relay trip reset (Fig 1).

\textsc\\
 1. refer to the diagram and wiring model to fix any wiring issues or missing components
